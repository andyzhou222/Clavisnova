# Clavisnova 快速操作清单

## 🎹 日常检查清单

### 每日检查
- [ ] 前端网站可正常访问
- [ ] 管理员后台可正常登录
- [ ] 数据提交功能正常（捐赠/需求表单）
- [ ] 导出功能正常工作

### 每周检查
- [ ] 查看新增捐赠数量
- [ ] 查看新增需求数量
- [ ] 检查数据导出功能
- [ ] 查看系统日志是否有错误

### 每月检查
- [ ] 导出完整数据备份
- [ ] 检查数据库存储使用情况
- [ ] 审核用户反馈和联系信息
- [ ] 清理过期或测试数据
- [ ] 检查邮件通知是否正常工作

## 🚀 快速操作步骤

### 查看统计数据
1. 访问管理员后台
2. 查看首页的统计数字
3. 记录捐赠和需求数量

### 导出本周数据
1. 进入 "Piano Registrations" 标签页
2. 点击 "Export to Excel"
3. 进入 "Requirements" 标签页
4. 点击 "Export to Excel"

### 处理用户联系
1. 进入 "Contacts" 标签页
2. 查看新联系信息
3. 回复用户邮件
4. 删除已处理的联系记录

### 清理无效数据
1. 在各标签页中搜索测试数据
2. 点击 "Delete" 删除无效记录
3. 确认删除操作

## 📊 数据监控指标

### 关键指标
- **捐赠转化率**: 新增捐赠数 / 页面访问量
- **需求满足率**: 已分配钢琴 / 总需求数
- **用户活跃度**: 每周活跃用户数

### 健康检查
- **API 响应时间**: < 2秒
- **数据库连接**: 正常
- **错误率**: < 5%

## 🆘 紧急处理

### 系统宕机
1. 检查 Render 应用状态
2. 查看错误日志
3. 重启应用（如果需要）
4. 通知用户系统维护

### 数据丢失
1. 从 Supabase 恢复备份
2. 检查数据完整性
3. 通知受影响用户

### 安全事件
1. 立即更改所有密码
2. 检查访问日志
3. 隔离受影响系统
4. 报告安全事件

## 📞 联系方式

- **技术支持**: 检查 GitHub Issues
- **用户反馈**: 通过平台联系表单
- **紧急情况**: 直接联系开发团队
